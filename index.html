FLL Scoring Demo App

Welcome to the demo-environment of the FLL scoring app developed by several Dutch FLL volunteers. This demo demonstrates several important features derivied from these requirements including the responsive design, the generation of the scoring file from an XML file, and options to communicate with other modules. DO NOT use this page during live events: everyone can manipulate the data and the environment will be reset every 30 minutes. However, feel free to experiment and test. If you want your own copy to be used at events, provide some feedback or check out other cool modules (like this display system visit our GitHub
